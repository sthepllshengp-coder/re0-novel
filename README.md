# 苍玄大陆 - 小说阅读网

一个本地化的 RE0 小说阅读网站，所有数据均存储在本地，无需联网即可阅读。

## 文件结构

```
苍玄大陆_网页版/
├── novel_reader_v2.html         # 主阅读器
├── chapter_index_full.json      # 章节索引
├── README.md                    # 使用说明
├── .gitignore                   # Git 忽略配置
└── novel_content/               # 小说内容目录
```

## 使用方法

### 本地预览

```bash
cd /Users/lishengping/Desktop/苍玄大陆_网页版
python3 -m http.server 8000
```

然后访问：`http://localhost:8000/novel_reader_v2.html`

### 部署到 GitHub Pages

1. 初始化 git 仓库
2. 推送到 GitHub
3. 在 GitHub 设置中启用 Pages

## 功能特点

1. **完全离线**：所有内容本地存储，无需联网
2. **主题切换**：白天/夜间/护眼三种模式
3. **字体调节**：四档可调
4. **章节目录**：按卷分类，可折叠展开
5. **搜索功能**：快速搜索章节
6. **阅读进度**：顶部进度条
7. **键盘快捷键**：左右键翻页，ESC 关闭面板
8. **响应式设计**：支持移动端
